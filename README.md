Bu proje, CourseApp adlı çok katmanlı bir .NET 8 uygulamasının hata ayıklama (debugging) ve kod iyileştirme çalışmasıdır.
Kod yapısı; Entity Framework Core, Repository ve Unit of Work tasarım desenleriyle oluşturulmuştur.
Projede amaç, mevcut kod tabanındaki hataları tespit etmek, mantıksal tutarsızlıkları gidermek, null kontrollerini eklemek ve mapping yapılarını düzenlemektir.

Amaç

Kodun derlenebilir, anlaşılır ve sürdürülebilir hale getirilmesi için refactoring ve hata temizleme işlemleri gerçekleştirilmiştir.
Bu proje yazılım hata ayıklama kategorisinde yarışma kapsamında hazırlanmıştır.
